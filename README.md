# Família Top Craft

### Passo a passo###

# Iniciando o servidor no terminal

# Copie e cole o comando abaixo no terminal para iniciar o serviço

/workspaces/minecraft-server/crafty/run_crafty.sh

# Ao lado do terminal, vá em "Portas" e clique no link segurando CTRL
# Em "Server" clique no servidor (Família Top Craft)
# Clique em "Start" (isso iniciará o servidor)

# Abra um novo terminal

# O próximo comando serve para gerar IP para outras pessoas conectarem no servidor

# Copie e cole o comando abaixo

playit

# Após rodá-lo, o Endereço de IP estará disponível

# O Endereço de IP do servidor Família Top Craft está abaixo

history-paradise.gl.joinmc.link

# Você pode entrar no minecraft e seguir os seguintes passos

Multijogador > Adicionar servidor > Endereço do Servidor (cole o endereço acima) > Concluído

# A partir daqui você pode entrar no servidor e jogar
# Divirta-se
